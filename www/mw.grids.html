mw.grids {GSIF}R Documentation

Malawi study area grids at 1/120 arcdegrees (1 km) resolution

Description

A list of soil covariates prepared for Malawi (94,215 grid nodes).

Usage

str(mw.grids)

Format

Data frame contains the following columns:

DEMSRT3

numeric; SRTM DEM

EV1MOD3

numeric; first principal component of the monthly MODIS EVI time series data

EV2MOD3

numeric; second principal component of the monthly MODIS EVI time series data

GLCESA3

factor; general land cover class based on the GlobCov v2.2

INSSRT3

numeric; potential incoming solar radiation derived in SAGA GIS

SLPSRT3

numeric; slope in degrees based on the SRTM DEM

TAXWRB3

factor; dominant soil class (group) following the World Reference Base soil classification system derived from the general soil polygon map of Malawi

TD1MOD3

numeric; first principal component of the 8-day MODIS day-time LST time series data

TN1MOD3

numeric; first principal component of the 8-day MODIS night-time LST time series data

TWISRT2

numeric; SAGA Topographic Wetness Index based on the SRTM DEM

x

numeric; x-coordinate in the UTM 36S coordiante system with WGS84 ellipsoid

y

numeric; y-coordinate in the UTM 36S coordiante system with WGS84 ellipsoid

Details

The landscape of Malawi is a series of dissected plateaus and mountainous areas along the Rift Valley i.e. lake Malawi. The size of the country is 118,000 square kilometers (the size of the rectangular block is 349 by 860 km). Dominat soil type map (TAXWRB3) was derived from the 1:600k scale soils map of Malawi and rasterized to 1 km resolution.

Note

Listed layer follow a standard naming convention used by WorlGrids.org (the standard 8.3 filename convention with at most eight characters): first three letter are used for the variable type e.g. DEM (digital elevation model); the next three letters represent the data source or collection method e.g. SRT (SRTM); the 6th character is the effective scale e.g. 3 indicates the 2nd standard scale i.e. 1/120 decimal degrees (about 1 km).

Examples

## Not run: # Load the data from GSIF page:
download.file("http://worldgrids.org/rda/mw.grids.rda", "mw.grids.rda")
load("mw.grids.rda")
gridded(mw.grids) <- ~x+y
proj4string(mw.grids) <- CRS(utm.csy)
# plot maps:
col = grey(rev(seq(0.1,0.975,0.025)))
names(mw.grids)
spplot(mw.grids[1], col.regions=col)
# write to a GIS format:
writeGDAL(mw.grids["EV1MOD3"], "EV1MOD3.tif", "GTiff", mvFlag=-99999)

## End(Not run)
