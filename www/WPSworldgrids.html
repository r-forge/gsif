WPSworldgrids {GSIF}R Documentation

Web Processing Service for worldgrids.org

Description

Overlay, subset and aggregation (wrapper) functions for the WPS worldgrids.org.

Usage

WPSworldgrids(layername, ...)

Arguments

layername

object

...

additional arguments

Details

WPS is implemented via the PyWPS module. The standard format for the images on the worldgrids.org repository is "GeoTiff". Complete grids (global coverage) can be obtained directly from worldgrids.org. We do not advise using the "bbox" argument to obtain grids that cover more than 30 percent of the global coverage.
